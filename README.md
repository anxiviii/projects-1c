Коллекция обработок, отчетов и доработок для различных конфигураций 1С:Предприятие 8.3.

## Содержимое

- Обработки для автоматизации рутинных задач.
- Отчеты для анализа и представления данных.
- Примеры интеграции с внешними сервисами.
- Учебные и экспериментальные проекты.

## Проекты

**Загрузка курсов ЦБ РФ:** получение курсов через HTTP и XML (HTTPСоединение, HTTPЗапрос, ЧтениеXML, клиент-сервер).

**ERP: автозаполнение «Отпустил»** — при создании документа «Реализация товаров и услуг» реквизит автоматически заполняется ответственным сотрудником склада из первой строки табличной части «Товары».

## Цель репозитория

Практика разработки на 1С, изучение возможностей платформы и накопление примеров решений для реальных задач.
